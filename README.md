### Prerequisites

- **Node.js:** `v18.20.0` or higher

### Installation

Clone the repository:

```sh
git clone https://github.com/MrRam-X/Virtual-Plant-Nursery-Client.git
cd planto-virtual-nursery


npm install
# OR
yarn install

npm run dev

```

